== Description ==

Used to display social media data fetched with [Feed Them Social](https://wordpress.org/plugins/feed-them-social/) backend and a carousel based on React and 

For further information and instructions please see on [Github](https://github.com/reneebeling/wp-feedThemSocial-react-extension).

== Installation ==

If you would prefer to do things manually then follow these instructions:

1. Upload the `wp-feedThemSocial-react-extension` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
